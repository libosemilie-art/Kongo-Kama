🚀 PROCHAINES ÉTAPES
════════════════════

1. Créer un fichier .env.local
   └─ Copier .env.example
   └─ Ajouter vos identifiants Supabase

2. Tester localement
   └─ npm install
   └─ npm run dev
   └─ npm run build

3. Initialiser Git & GitHub
   └─ git init
   └─ git add .
   └─ git commit -m "Initial commit"
   └─ Créer repo sur GitHub
   └─ git push origin main

4. Connecter Netlify
   └─ Aller sur netlify.com
   └─ New site > Import from Git
   └─ Sélectionner le repo
   └─ Ajouter variables d'environnement
   └─ Déployer!


📊 VARIABLES D'ENVIRONNEMENT
════════════════════════════

Vous aurez besoin de:
  • VITE_SUPABASE_URL = https://xxxxx.supabase.co
  • VITE_SUPABASE_ANON_KEY = eyJhbGc...

Où les ajouter:
  1. .env.local (pour développement local)
  2. Netlify Dashboard > Site settings > Environment
